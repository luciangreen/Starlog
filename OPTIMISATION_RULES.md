# OPTIMISATION_RULES

## Stage 11 category split

The optimisation documentation distinguishes these categories:

1. accumulator rewrites,
2. polynomial discovery by Gaussian elimination,
3. general indexed-variable tracing,
4. reduction to pattern matching plus irreducible commands,
5. formula reconstruction from first principles,
6. indexed-variable formula derivation,
7. direct indexed rule generation.

## Design rule on structural names

Named structural labels (for example diagonal, row, or column wording) are not optimisation categories.

If such shapes appear, they are treated as outcomes of the same first-principles tracing and reconstruction pipeline, not as hardcoded optimisation classes.
